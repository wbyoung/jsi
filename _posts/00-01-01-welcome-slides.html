Command Line

Let's look at basic commands

The command line can be intimidating, that's okay.

pwd

Print the current working directory

ls [path]

List the contents of a directory (or the current directory)

cd <path>

Change the current working directory

cd ~

Go to your home directory

cd ..

The .. indicates the parent directory

cp <src> <dest>
mv <src> <dest>
rm <path>

Copy, move, and remove files

man <command>

Get “help”

Man pages are often very difficult to understand.

How do I …?

Ask.

You can do just about anything on the command line.

Learning how to use the command line will make you more productive.

JavaScript History

JavaScript History

ECMAScript History

  • ECMAScript is the language specification.
  • Implementations originally included JavaScript and JScript for Mozilla and Microsoft's browsers respectively.

Browser Uses

  • 1995 – Netscape Navigator
  • 1996 – Internet Explorer
  • 1996 – Dynamic HTML (DHTML)
  • 1997 – ECMAScript Standardized
  • 2004 – Web 2.0
  • 2005 – AJAX
  • 2006 – jQuery
  • 2010 – Backbone.js
  • Browser wars saw quick addition of JS in both browsers
  • Gmail was basically the start of Web 2.0
  • Prototype.js, Dojo were a little before jQuery in 2005

Server Uses

  • 1990s – Netscape & Microsoft
  • 2009 – Node.js
  • Note how nothing happened in the 90s with this

Caution

JavaScript is still misunderstood by many developers.

Some things you read may be wrong.

Because of the history of only being in the browser, for a long time JavaScript wasn't seen as a particularly useful or powerful language. That was never actually true, but it's probably that perception and a lack of a singular source of documentation that has shaped today's environment.

What does this mean?

Expressing Meaning


// given two points
var firstPointX = -2;
var firstPointY = 1;
var secondPointX = 2;
var secondPointY = 3;

// determine the slope
var deltaY = secondPointY - firstPointY;
var deltaX = secondPointX - firstPointX;
var slope = deltaY / deltaX;

// find the y-intercept
var b = firstPointY - slope * firstPointX;

// show the slope-intercept form of the equation
console.log('y = ' + slope + 'x + ' + b);

How about this?

Ignore the new syntax here & focus on the intention expressed.

   over multiple lines */

Commenting is an art form that you'll soon master.

Variables

Variables contain or point to values.

They allow us to break down our code and better express intent.

The idea of a variable pointing to a value comes from other programming languages, but the terminology is used frequently among all programmers including JavaScript programmers.

Variables Can Be Changed


var string;
string = "irrelevant value";
string = "hello";
string = string + " ";
string += "world";
console.log(string);
